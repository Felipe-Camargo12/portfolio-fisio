# Portfolio Fisioterapeuta Geovana Santos - Landing Page

![Hero do Projeto](./public/Modelo%20-%20Fisio.png)

## 📋 Sobre o Projeto

Página web desenvolvida para a **Fisioterapeuta Geovana Santos**, com foco em apresentar seus serviços e facilitar o contato com os pacientes. O design foi construído para transmitir profissionalismo e cuidado, guiado pelo lema *"Reabilitação e Movimento com Propósito"*.

O site atua como um cartão de visitas digital completo, permitindo que os visitantes conheçam o histórico da profissional, entendam os serviços oferecidos e realizem agendamentos de forma rápida e intuitiva.

## ✨ Funcionalidades

* **Apresentação e Bio:** Seção "Olá! Eu sou a Fisioterapeuta Geovana Santos", detalhando a trajetória e especializações da profissional.
* **Serviços e Especialidades:** Exibição clara das áreas de atuação e tratamentos oferecidos.
* **Call-to-Action (CTA):** Botões estratégicos de "Agendar sua avaliação" integrados na página.
* **FAQ e Suporte:** Seção de dúvidas frequentes para otimizar o tempo de atendimento.
* **Layout Responsivo:** Experiência de navegação fluida em smartphones, tablets e desktops.

## ⚙️ Tecnologias Utilizadas

- **React** — Biblioteca para construção da interface
- **Tailwind CSS** — Estilização rápida com classes utilitárias
- **JavaScript** / **TypeScript**
- **Vite** (ou CRA, conforme o setup)

## Como usar

```bash
npm install
npm run dev
```
